# 1. STRAIGHTFORWARD PROMPTS

## Definition

Straightforward prompting directly asks the AI system to perform a task without additional context or formatting.

## Examples

### Prompt 1

“Define Artificial Intelligence in one sentence.”

### Generated Response

Artificial Intelligence is the simulation of human intelligence processes by machines and computer systems.

---

### Prompt 2

“List three advantages of online learning.”

2. Access to global educational resources

# 3. PRECEDING QUESTION PROMPTING

## Definition

Preceding Question Prompting involves asking an introductory or guiding question before the main question to improve context understanding.

## Examples

### Prompt 1

“Why is cybersecurity important? Explain how firewalls help protect computer networks.”

### Generated Response

Cybersecurity is important because it protects systems, data, and networks from unauthorized access and cyberattacks. Firewalls help protect networks by monitoring incoming and outgoing traffic and blocking suspicious activities.

---

### Prompt 2

“How does cloud computing benefit businesses? Describe the advantages of cloud storage.”

### Generated Response

Cloud computing benefits businesses by reducing infrastructure costs and improving accessibility. Cloud storage allows secure data access from anywhere and supports easy backup and recovery.

---

## Evaluation

| Criteria              | Observation |
| --------------------- | ----------- |
| Accuracy              | High        |
| Coherence             | Excellent   |
| Context Understanding | Excellent   |
| User Experience       | Very Good   |

# COMPARISON OF PROMPT TYPES

| Prompt Type                  | Main Advantage               | Limitation                   |
| ---------------------------- | ---------------------------- | ---------------------------- |
